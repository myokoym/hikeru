# Hikeru

Hikeru is a music synthesizer by Ruby/GStreamer and Ruby/GTK2.

## Dependencies

* Ruby/GStreamer and Ruby/GTK2 in
  [Ruby-GNOME2](http://ruby-gnome2.sourceforge.jp/)

## Installation

    $ gem install hikeru

## Usage

### Launch

    $ hikeru
